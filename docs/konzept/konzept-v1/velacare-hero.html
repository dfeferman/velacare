Copywriting — Warum so
  • Headline: Produkt + kostenlos + Lieferversprechen — eine Zeile, ohne Komma-Pause
  • Terrakotta-Em: Das Versprechen „jeden Monat nach Hause" ist das Herzstück
  • Subtext-Struktur: Frage → Antwort → Entlastung — räumt Unsicherheit in einem Satz aus
  • CTA: „kostenlos & unverbindlich" eliminiert Angst vor Vertragsfalle
  • Reassurance-Zeile: 3 Einwände direkt unter dem Button abgefangen
Bildbrief für Fotograf
  • 2 Personen: Angehörige (40–60J) + ältere Person (70+J), entspannt, gemeinsam
  • Ort: helles Wohnzimmer oder Küche — kein Krankenhaus, kein Pflegebett sichtbar
  • Stimmung: warm, Tageslicht, natürlich — kein Stock-Photo-Feeling
  • Keine Pflegehilfsmittel im Bild — das Gefühl kommt vor dem Produkt
  • Format: Hochformat 4:5 — passt zur organischen Frame-Form
Trust Bar — Reihenfolge
  • 1. Was bekomme ich? → 42€/Monat
  • 2. Was kostet es mich? → 0€ Zuzahlung
  • 3. Bin ich gebunden? → Keine Bindung
  • 4. Funktioniert es bei mir? → Deutschlandweit
  • Reihenfolge = Entscheidungsweg des Nutzers
Technische Hinweise
  • Hero-Höhe: min-height: calc(100vh - 54px)
  • Photo-Frame: organischer border-radius, keine SVG-Clip nötig
  • Floating Elemente: animation-delay gestaffelt (.7s, .9s)
  • Eyebrow-Dot: @keyframes pulse — signalisiert Aktualität
  • CTA Shadow verstärkt sich auf :hover — Click-Affordance
